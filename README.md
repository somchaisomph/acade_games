<h2>Components:</h2>
<ol>
<li>Raspberry PI Model B3 or B2 or B2 Plus</li>
<li>2 axis joystick</li>
<li>10K Ohm resistor</li>
<li>MCP3208</li>
<li>Jump wires</li>
</ol>
<hr/>
<h2>Dependencies</h2>
<h3>Install Python SpiDev</h3>
<ol>
<li>$ git clone https://github.com/doceme/py-spidev</li>
<li>$ cd py-spidev</li>
<li>
 $ sudo python setup.py install
 
</li>
</ol>
